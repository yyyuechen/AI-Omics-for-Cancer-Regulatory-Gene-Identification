# AI-Omics-for-Cancer-Regulatory-Gene-Identification
Reproduction of a published paper on AFB1-induced hepatocellular carcinoma, integrating transcriptomic differential expression, machine learning, network toxicology, and molecular docking to identify candidate regulatory genes and key target proteins.

# Project overview
This repository contains scripts and workflows for reproducing a published study on aflatoxin B1 (AFB1)-induced hepatocellular carcinoma (HCC). The analysis integrates differential expression analysis across multiple transcriptomic datasets, machine learning-based feature selection, network toxicology, and molecular docking to identify candidate target genes and core regulatory genes associated with AFB1-related HCC, and to assess the binding specificity of AFB1 toward the predicted target proteins.
